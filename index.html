Um pouco sobre mim

Sou Desenvolvedor Full Stack com foco no desenvolvimento de aplicações web completas, desde a construção da interface até a lógica de negócio e integração com banco de dados.

Trabalho com HTML, CSS e JavaScript no front-end, além de Java e Spring Boot no back-end, desenvolvendo APIs e sistemas estruturados. Também possuo experiência com SQL e MySQL, além de utilizar Docker para criação de ambientes de desenvolvimento e execução de aplicações de forma consistente e organizada.

Participei do desenvolvimento de projetos full stack em equipe, incluindo um sistema de agendamento de consultas médicas e um gerenciador de eventos, experiências que fortaleceram minhas habilidades em desenvolvimento colaborativo, organização de código e resolução de problemas.

Atualmente estou finalizando um gerenciador de tarefas como projeto pessoal, com o objetivo de aprofundar meus conhecimentos na construção de aplicações completas e bem estruturadas.

Sou estudante de Engenharia de Software na Univille (Joinville), atualmente no 7º semestre, com previsão de conclusão em junho de 2027. Durante a graduação venho consolidando minha base em arquitetura de software, programação orientada a objetos e desenvolvimento de sistemas.

Busco uma oportunidade de estágio em desenvolvimento de software, onde eu possa contribuir com a criação de produtos e soluções enquanto continuo evoluindo tecnicamente. Também tenho interesse em áreas relacionadas como DevOps, QA e suporte, pois acredito que compreender diferentes etapas do ciclo de desenvolvimento torna um profissional mais completo.

Sou dedicado, tenho facilidade para trabalhar em equipe e estou em constante evolução, sempre investindo em estudos e projetos práticos para aprimorar minhas habilidades como desenvolvedor.
