MedTime

Projeto da disciplina de Fábrica de Software com o MedTime, um sistema web de agendamento médico que conecta pacientes e profissionais da saúde.

Funcionalidades
Cadastro e Login de Usuários: Suporte para médicos e pacientes.
Gestão de Especialidades: Cadastro e visualização de especialidades médicas.
Gestão de Médicos: Cadastro de médicos com informações detalhadas.
Agendamento de Consultas: Sistema completo de agendamento.
Interface Responsiva: Design moderno e adaptável.

Tecnologias Utilizadas
Backend: Java 17, Spring Boot 3.x
Frontend: HTML, CSS, JavaScript
Banco de Dados: MySQL/PostgreSQL
ORM: JPA/Hibernate
Build Tool: Maven
Containerização: Docker

No projeto, atuei no desenvolvimento do frontend junto com o Julia Lopes enquanto a Isabele Pires e o Vitor Hugo Ramos foram responsáveis pelo backend.

O trabalho em equipe foi essencial para que tudo saísse como planejado. Agradeço à equipe pelo comprometimento, colaboração e consistência ao longo de todo o projeto!

Tasky

Projeto pessoal de Gerenciamento de Tarefas, desenvolvido com o objetivo de aprofundar conhecimentos em desenvolvimento web e aprimorar minhas habilidades na construção de aplicações completas, desde o backend até o deploy em produção.

Funcionalidades
Cadastro e Login de Usuários: Sistema de autenticação para acesso à plataforma. Gestão de Usuários: Possibilidade de visualizar e alterar informações do usuário. Gestão de Tarefas (CRUD): Criação, visualização, edição e exclusão de tarefas. Organização de Atividades: Permite acompanhar e gerenciar tarefas de maneira prática. Interface Responsiva: Layout adaptável para diferentes dispositivos.

Tecnologias Utilizadas
Backend: Java + Spring Boot para desenvolvimento da API.
Banco de Dados: MySQL para persistência e gerenciamento dos dados.
Frontend: HTML, CSS e JavaScript no desenvolvimento da interface.
Containerização: Docker para empacotamento da aplicação.
Deploy Backend: Microsoft Azure utilizando containers.
Deploy Frontend: Vercel para hospedagem da interface.
Testes de API: Postman para validação e testes das requisições.

Neste projeto, fui responsável por todo o desenvolvimento da aplicação, incluindo a implementação das funcionalidades, integração entre frontend e backend, configuração do banco de dados, containerização com Docker e deploy em nuvem.

O projeto foi desenvolvido com foco em estudo e aperfeiçoamento técnico, permitindo colocar em prática conceitos de arquitetura de aplicações, APIs REST, integração de serviços e publicação de aplicações em ambiente de produção.
